Définition 1.13.

On appelle probabilité de l’espace \((\Omega,\mathscr T)\) toute application \(\Pr:\mathscr T\to[0,1]\) telle que :
  • \(\Pr{\Omega}=1\) ;
  • si \((A_n)_{n\in\N}\) est une suite d’éléments de \(\mathscr T\) deux à deux disjoints alors
    \begin{equation*} \Pr{\bigcup\limits_{n=0}^\infty A_n}=\sum\limits_{n=0}^\infty \Pr{A_n}\text{.} \end{equation*}
Si \(\PP\) est un probabilité de \((\Omega,\mathscr T)\) alors le triplet \((\Omega,\mathscr T,\PP\) est dit un espace probabilisé.
